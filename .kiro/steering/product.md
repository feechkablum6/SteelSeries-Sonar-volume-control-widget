# Product Overview

StellGG Volume Mixer Widget — desktop-виджет для Windows, управляющий громкостью аудиоканалов SteelSeries Sonar.

## Основные возможности
- Прозрачное окно с glassmorphism эффектом (frosted glass)
- 6 независимых каналов: Master, Game, Chat, Media, Aux (Music), Mic
- Интерактивные вертикальные слайдеры громкости
- Кнопки mute с визуальной индикацией
- Drag-and-drop перемещение окна
- Автоскрытие при перекрытии другими окнами
- Сохранение позиции между сессиями
- Интеграция с SteelSeries Sonar через native-sound-mixer

## Целевая аудитория
Геймеры и пользователи SteelSeries GG, которым нужен быстрый доступ к управлению звуком.

## Ключевые требования
- Компактный размер (540x300px)
- Премиальный внешний вид
- Минимальное отвлечение от основной работы
- Windows 10/11 совместимость
- Always-on-top с умным скрытием
